# Auto Water 🌊

> English | [中文](#auto-water-1)

A **Stardew Valley** mod that automatically waters all your crops every morning. No more manual watering!

## Features

- 🌱 Waters **all crops** on your farm automatically at the start of each day
- 🏠 Supports **Greenhouse** watering
- 🏝️ Supports **Ginger Island** farm watering
- 🪴 Supports **Garden Pots** (indoor pots)
- 🔔 Optional HUD notification showing how many tiles were watered
- ⚙️ Fully configurable (toggle each feature on/off)

## Configuration

After first run, edit `config.json`:

```json
{
  "EnableAutoWater": true,      // Master switch
  "ShowNotification": true,      // Show watered count
  "OnlyWaterCrops": true,        // Skip empty tilled soil
  "WaterGreenhouse": true,       // Water greenhouse crops
  "WaterGingerIsland": true,     // Water Ginger Island crops
  "WaterGardenPots": true        // Water indoor garden pots
}
```

## Install

1. Install [SMAPI](https://smapi.io) (4.0+)
2. Download this mod
3. Extract to `Stardew Valley/Mods/AutoWater/`
4. Start the game!

## Compatibility

- ✅ Stardew Valley 1.6+
- ✅ Single-player & Multiplayer
- ✅ Compatible with other farming mods

# Auto Water 自动浇水 🌊

一个 **星露谷物语** mod，每天早上自动浇灌全农场所有作物，告别手动浇水！

## 功能

- 🌱 每天早晨**自动浇灌**农场所有作物
- 🏠 支持**温室**浇水
- 🏝️ 支持**姜岛**农场浇水
- 🪴 支持**花园盆栽**浇水
- 🔔 可选的 HUD 提示，显示浇灌了多少格
- ⚙️ 高度可配置（每个功能可独立开关）

## 配置

首次运行后编辑 `config.json`

## 安装

1. 安装 [SMAPI](https://smapi.io) (4.0+)
2. 下载本 mod
3. 解压到 `Stardew Valley/Mods/AutoWater/`
